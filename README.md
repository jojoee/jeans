Simple and clean frontend toolkit that you can wear it everywhere

## Getting Started
1: Install via `npm`
```
1.1 Install npm
1.2 Install package: `npm install --save jeans-kit`
```
2: Install via `bower`
```
2.1 Install npm
2.2 Install bower: `npm install -g bower`
2.3 Install package: `bower install --save jeans-kit`
```
